##斐波那契数列

计算斐波那契数列的第 n 个数字。

斐波那契数列的定义是：第一个和第二个数字为 1，之后的每个数字都是前两个数字之和。因此，斐波那契数列的前几个数字是 1, 1, 2, 3, 5, 8, 13, 21, ... 依此类推。

###项目说明：

>src目录下为题目代码源文件，需要完成的内容为fibonacci.py里的get_num方法。
>src目录下的__init__.py文件不需要改动。



### 示例：
>注意：不需要进行输入输出的处理，输入作为函数的参数传入，结果直接作为函数的返回值返回。

	输入：1
	输出：1
	
	输入：2
	输出：1
	
	输入：3
	输出：2
